# Portfolio site

A single-page portfolio — open index.html in any browser. No Node.js required.

## Stand out features
- Command palette (Ctrl+K)
- Skills with proof of use
- Bento project grid with impact metrics
- Light/dark theme

## Quick start
1. Edit data.js with your info
2. Open index.html in Chrome or Edge

## Deploy
- GitHub Pages or Netlify Drop (drag folder to app.netlify.com/drop)
